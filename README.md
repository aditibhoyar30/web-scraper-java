# Web Scraper for Headlines (Java)

## Description
A Java-based web scraping application that fetches and displays
live news headlines using the Jsoup library.

## Technologies Used
- Java
- Jsoup
- HTTP Protocol
- CSS Selectors

## Features
- Fetches live headlines from a website
- Uses browser-like headers to avoid blocking
- Parses HTML using CSS selectors
- Handles exceptions gracefully
